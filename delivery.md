## How to Run/Deploy
1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Start the backend:**
   ```sh
   node server.js
   ```
   - Runs on port 3001 by default
3. **Start the frontend:**
   ```sh
   npm start
   ```
   - Runs on port 3000 by default
4. **Run tests:**
   ```sh
   npm test
   ```
